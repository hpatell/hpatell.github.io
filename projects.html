Projects

Garden Project

This project is a software application designed for simulating an interactive garden where the user decides what native plants are most supported by the lepadotris through the input of the conditions of the garden the user modifies and how much the user's budget is.

Stocks Project

stocks4me is a website that allows you to create a curated stock dashboard in which you can modify with a custom wallet size and watch your stocks grow. The dashboard is customizable, you can search to add more. As you add stocks, news stories about your stocks become curated in the news tab.

Shell Project

This project is a Unix shell that is simple, creates processes, handles signals, uses Unix system calls and C library function calls. Features include background processes, file redirection, pipes, and programming with POSIX threads (pthreads).
